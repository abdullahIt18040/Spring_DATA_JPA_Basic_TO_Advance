## Tomcate
Apache Tomcat হলো একটি Java Web Server + Servlet Container, যা দিয়ে Java (Spring, JSP, Servlet) অ্যাপ রান করা হয় .thread contain into tomcat । Apache Tomcat প্রতিটা request handle করার জন্য thread ব্যবহার করে।
